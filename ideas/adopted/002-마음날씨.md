# 마음날씨 (MaumNalssi)

## 한줄 요약
> 한국어 감정 어휘로 매일 감정을 기록하고, AI가 패턴을 분석하여 맞춤형 셀프케어 행동을 제안하는 감정 관리 앱

## 해결하려는 문제 (Pain Point)
- 감정 기록 앱(하루콩, MOODA)은 "기록하고 끝" — 기록 후 "그래서 어떻게 해야 하는가?"에 대한 답 없음
- 심리상담 앱(마인드카페, 트로스트)은 전문 상담 중심 — 3~10만원/회 비용 부담 + 심리적 진입 장벽 높음
- "기록과 상담 사이"의 중간 지대 부재: 전문 상담은 부담스럽지만, 단순 기록보다 더 능동적인 관리가 필요한 대다수 사용자층
- 한국어 고유 감정 표현('서운하다', '답답하다', '억울하다', '눈치 보인다') 미반영 — 기존 앱은 영어권 감정 분류 체계를 번역한 수준
- 2026 필코노미 트렌드: "기분을 관리하는 것"이 하나의 역량이자 소비 동기가 되는 시대. 감정 인식 → 패턴 발견 → 대응 행동의 완결된 루프 필요

## 타겟 사용자
- **주요 타겟**: 감정 기록은 하고 싶지만 전문 상담까지는 부담스러운 MZ세대
- **연령대**: 10대 후반 ~ 30대 중반
- **규모 추정**: 하루콩 글로벌 1,000만+DL (한국 비중 20% → 약 200만), 마인드카페 400만+, 트로스트 80만+. 감정 관리 앱 사용 경험자 약 500만 명 이상. AI 심리상담 이용 의향 40% (한국리서치 2025)

## 핵심 기능 (MVP 범위, 최대 3개)
1. **한국어 감정 태깅 (10초 기록)**: 한국어 특화 감정 어휘 30-40종 제공 (서운함, 답답함, 억울함, 뿌듯함, 설렘 등). 터치로 감정 태그 1-3개 선택 + 선택적 한줄 메모. 활동 태그 연결 (출근, 회의, 운동, 식사, 대화 등)
2. **AI 감정 패턴 리포트 (주간/월간)**: 감정-시간-활동 상관관계 분석. "화요일 오후에 '답답함'이 집중됩니다", "운동한 날은 긍정 감정이 2배 높습니다" 등. 태그 기반 통계 분석 (NLP/ML 불필요, Supabase SQL로 구현)
3. **감정별 셀프케어 행동 카드**: 현재 감정 상태에 맞는 실행 가능한 행동 제안 (5분 호흡법, 산책, 좋아하는 음악 듣기, 따뜻한 차 한잔 등). 사용자 피드백(도움됨/아님)으로 제안 정확도 점진 개선. 규칙 기반 매핑 + 향후 LLM API로 개인화 확장 가능

## 수익 모델
- **프리미엄 구독** (월 3,500원 / 연 27,900원):
  - 무료: 일일 감정 기록, 기본 주간 요약, 셀프케어 카드 3개/일
  - 유료: AI 상세 패턴 분석 리포트, 월간/분기별 감정 트렌드, 무제한 셀프케어 카드, 감정-활동 상관관계 인사이트, 데이터 내보내기(PDF)

## 기술 구현 검토
- **React Native + TypeScript**: 완전 구현 가능. 감정 태그 UI (칩/태그 컴포넌트), 차트/그래프 (Victory Native 등), 캘린더 뷰 등 표준 라이브러리로 충분
- **Supabase**: 감정 기록 테이블 (user_id, emotion_tags, activity_tags, memo, timestamp), PostgreSQL 집계 함수로 패턴 분석 쿼리 구현. RLS로 사용자별 데이터 격리. Edge Functions로 주간 리포트 생성 가능
- **외부 서비스 의존도**: 낮음. 핵심 분석은 태그 기반 SQL 통계로 구현 (AI/ML 모델 불필요). 푸시 알림(Expo Push), 인앱결제(RevenueCat). 향후 확장 시 LLM API(OpenAI)로 개인화 인사이트 문구 생성 가능하나 MVP에서는 선택사항

## 판정
- **결과**: `채택` (24점 이상)
- **사유**: 총점 24점으로 채택 기준(24점) 경계선 충족. 기술 실현성(5점)과 MVP 가능성(5점)이 최고점으로, 1인 개발에 최적화된 프로젝트. "기록→분석→행동"의 완결 루프는 기존 경쟁사에 없는 유효한 차별점. 필코노미/메타센싱 트렌드와의 시의성, 한국 기분 기록 앱 사용자 4배 성장(2023 50만→2025 200만, 와이즈앱)도 긍정적 신호. 다만 경쟁 강도(3점)와 수익 모델(3점)에서 리스크 존재: (1) 라일리 하루(2024.12 출시, 자체 LLM 멀티모달 감정분석)가 가장 유사한 직접 경쟁자로 부상, (2) 블루시그넘(하루콩)이 라임 AI 개발+하버드 MGH 공동연구로 AI 확장 준비 중, (3) 감정 기록 앱의 유료 전환율 불확실 (건강&피트니스 앱 DL→유료 9.4%이나 한국 시장은 더 낮을 가능성). 채택하되, PRD 단계에서 (1) 라일리 하루 대비 차별화 전략(행동 제안 중심 vs 분석 중심), (2) 수익 모델 강화(프리미엄+무료체험, 캐릭터/테마 판매 병행), (3) 데이터 축적 기반 moat 전략을 구체화해야 함.
