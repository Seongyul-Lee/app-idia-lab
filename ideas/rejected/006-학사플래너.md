# 학사플래너 (AcadeMate)

## 한줄 요약
> 시간표·과제 마감일·시험 일정을 한 곳에서 관리하고, 놓치기 쉬운 마감을 똑똑한 알림으로 챙겨주는 대학생 전용 학사일정 플래너

## 해결하려는 문제 (Pain Point)
- **과제·시험 마감일 분산 관리 문제**: 대학생은 LMS(e-class), 카카오톡, 에브리타임, 구글캘린더, 노션 등 여러 도구에 일정을 흩어놓고 관리하여 마감일을 놓치는 경우가 빈번함
  - 네이버 블로그 다수 게시글에서 "과제 마감일 깜빡", "시험 일정 놓침"이 대학생 스트레스 주요 원인으로 반복 언급
  - 동신대학교 공식 블로그: "대학생이 되면 시간표, 과제 마감일, 팀플 일정, 개인 약속까지 한꺼번에 모두 관리해야하는데" 기존 앱이 학사일정에 특화되지 않아 불편함을 호소
  - 충북보건과학대 블로그: 할 일 관리, 공부 집중, 시험·과제 일정 관리 도구가 산재되어 있어 "대학생활 리듬을 잡기 어렵다"고 지적
- **에브리타임의 한계**: 에브리타임은 시간표·커뮤니티 중심이며, 캘린더 기능은 자체적으로도 "아직 MVP 단계라 부족한 점이 있다"고 인정(비누팀 공식 블로그, 2025.12). 과제 마감일 추적·D-day 알림·우선순위 관리 기능이 없음
- **범용 앱의 학사 맥락 부재**: 구글캘린더·노션은 범용 도구라 '과목별 과제 추적', '시험 D-day 카운트다운', '학점 시뮬레이션' 같은 학사 특화 기능이 없어 대학생이 직접 템플릿을 만들어야 하는 번거로움

## 타겟 사용자
- **주요 타겟**: 4년제/전문대 재학생 (특히 신입생~2학년, 복학생)
- **연령대**: 18~26세
- **규모 추정**: 한국 대학 재학생 약 280만 명. 에브리타임 가입자 800만+. 타겟 DAU 시간표 관리 수요는 매 학기 수강신청 시즌(연 2회) 폭증. 초기 타겟 10만 다운로드 현실적

## 핵심 기능 (MVP 범위, 최대 3개)
1. **과목별 과제·시험 일정 관리**: 수강 과목을 등록하면 과목별로 과제·시험·발표 일정을 입력하고, D-day 카운트다운 + 맞춤 알림(1일 전, 3일 전 등) 제공. 과제 상태(미시작/진행중/완료) 추적
2. **주간 시간표 + 일정 통합 뷰**: 수업 시간표 위에 과제 마감일·시험·개인 일정을 오버레이하여 한눈에 "이번 주에 뭘 해야 하는지" 파악. 공강 시간에 과제 시간 자동 제안
3. **학점 시뮬레이터**: 현재까지의 성적을 입력하면 목표 학점(졸업 요건) 달성을 위해 남은 학기에 필요한 평균 성적을 계산. 이수 학점 현황 대시보드

## 수익 모델
- **프리미엄 구독(월 1,900~2,900원)**: 무제한 과목 등록, 위젯 커스터마이징, 반복 과제 자동 생성, 통계/분석 리포트
- **무료 기본 플랜**: 5과목까지 무료, 기본 알림, 시간표 뷰
- **대학교 제휴 광고(장기)**: 대학 주변 카페·서점·스터디카페 등 타겟 광고

## 기술 구현 검토
- **React Native + TypeScript**: 완전 구현 가능. 캘린더 UI(react-native-calendars), 로컬 알림(react-native-push-notification/notifee), 위젯(react-native-widget) 등 성숙한 라이브러리 존재
- **Supabase**: 유저 인증(소셜 로그인), 과목·과제·시험 데이터 CRUD, 구독 상태 관리에 충분. Row Level Security로 개인 데이터 보호. 실시간 동기화(Realtime) 활용 가능
- **외부 서비스 의존도**:
  - 푸시 알림: Firebase Cloud Messaging (필수, 무료)
  - 결제: 인앱결제(Google Play/App Store 기본 제공)
  - 별도 외부 API 의존 없음 → 1인 개발 적합

## 판정
- **결과**: 탈락
- **탈락 단계**: 2단계 (사업성/기술/경쟁력 평가)
- **미달 항목**: 경쟁 강도 (1/5점), 수익 모델 명확성 (2/5점)
- **사유**: 에브리타임이 2025년 10월 캘린더 기능을 정식 출시하고 학사 특화 기능 확장을 공식 예고한 상황에서, 본 아이디어의 핵심 차별점(과제 D-day 알림, 상태 추적, 시간표 통합 뷰)이 에브리타임에 흡수될 가능성이 극히 높음. MAU 300만·전국 400개 캠퍼스 데이터·커뮤니티 시너지 등 에브리타임의 네트워크 효과와 데이터 해자를 1인 개발 독립 앱으로 극복하는 것은 비현실적. 또한 에브리타임이 무료로 유사 기능을 제공하므로 월 1,900~2,900원 구독 유료 전환이 매우 어려움. 학점 시뮬레이터는 웹에 무료 대안 다수 존재하며 학기당 1~2회 사용으로 단독 결제 트리거 불충분.
- **날짜**: 2026-02-01
