## 직접 경쟁자 분석

### 1. 파프리카케어 (PapricaCare) — 한국

| 항목 | 내용 |
|------|------|
| 개발사 | Sphere Corp. (구 Onions, 라이프시맨틱스에 2024년 인수) |
| 누적 회원 | 23만 명 (2022년 기준), 다운로드 45만 회 |
| 앱스토어 평점 | 3.6/5 (123개 평가, iOS) |
| 가격 | 무료 |
| 핵심 기능 | 처방전/약봉투 OCR 인식, 의약품 정보 조회, 복약 알림, 가족 프로필 추가 |

**강점:**
- 처방전·약봉투 촬영으로 자동 약 등록 (OCR)
- 건보심사평가원 의료 빅데이터 기반 질병·약 통계 정보 제공
- DUR(병용금기) 주의사항 알림
- "가족 추가" 기능으로 한 기기에서 여러 가족 프로필 관리 가능

**약점:**
- 앱스토어 리뷰에서 **복약 알림이 안 오는 오류** 반복 보고 (2020~2021년)
- "가족 추가"는 **동일 기기 내 프로필 전환**이지, 원격 모니터링/미복용 알림 전송이 아님
- 수동 입력 기능이 뒤늦게 추가됨 (초기 미지원)
- 라이프시맨틱스 인수 후 라이프레코드 플랫폼 통합 방향 → 복약관리 단독 앱으로서의 집중도 약화 가능성
- 평점 3.6으로 낮은 편, 리뷰 수도 123개로 적음

**차별 기회:**
- 원격 가족 연동(미복용 시 자녀에게 푸시 알림)은 미지원 → 복약케어의 핵심 차별점
- 어르신 전용 대형 UI 부재
- 알림 안정성 문제 → 신뢰도 확보 기회

### 2. 네이버 헬스케어 복약관리 — 한국 (2025.11 출시)

| 항목 | 내용 |
|------|------|
| 개발사 | 네이버 |
| 사용자 수 | 미공개 (네이버앱 내 서비스, 잠재 도달 수천만) |
| 가격 | 무료 (네이버앱 내장) |
| 핵심 기능 | 약 봉투 OCR 촬영 → 자동 등록, 복용 여부·복용률 확인, 날짜별 복약 내역·증상 기록 |

**강점:**
- 네이버앱 내장으로 **별도 설치 불필요**, 기존 4,000만+ MAU 기반
- AI/OCR 기술력 (네이버 클로바)
- 네이버 헬스케어 생태계 (만보기, 병원예약, 증상체크, 실손보험) 통합
- 지속적 고도화 예고 (진료예약-EMR 연동 등)

**약점:**
- **가족 연동/원격 모니터링 기능 없음** (2025.11 출시 시점 기준, 개인 복약관리에 한정)
- 네이버앱 내 서브 서비스 → 전용 앱 대비 접근 경로 복잡 (검색 또는 전체서비스 진입)
- 어르신 전용 UX 최적화 미확인 (네이버앱 기본 UI 그대로)
- 독립적 푸시 알림 체계 불명확 (네이버앱 알림에 묻힐 가능성)

**위협 수준: 중~높음**
- 네이버가 향후 가족 공유 기능을 추가할 가능성 존재
- 다만 네이버 헬스케어는 범용 건강관리 플랫폼이므로 "가족 연동 복약 특화"로 깊이 파고들 가능성은 낮음
- 네이버는 B2C 복약관리보다 B2B(병원·약국·보험) 생태계에 집중 추세

### 3. Medisafe — 글로벌

| 항목 | 내용 |
|------|------|
| 개발사 | Medisafe (미국/이스라엘) |
| 누적 사용자 | 700만+ (2021년 기준) |
| 앱스토어 평점 | 4.7/5 |
| 가격 | 프리미엄 $4.99/월 또는 $39.99/년 (2026.1부터 유료 전환) |
| 핵심 기능 | 복약 알림, Medfriend(가족 알림), 병용금기 체크, AI 예측 알림 |

**강점:**
- **Medfriend 기능**: 가족/친구에게 미복용 알림 전송 → 가족 연동의 글로벌 선두
- 글로벌 700만 사용자, 평점 4.7로 시장 검증 완료
- 2026년 유료 전환 → 수익 모델 전환 사례
- 제약사 B2B 파트너십 (Eisai, Biogen 등)
- 평균 복약 순응도 86% 달성 (업계 평균 50% 대비)

**약점:**
- **한국어 미지원**, 한국 의약품 DB 미연동
- 한국 처방전/약봉투 OCR 불가
- 2026년 유료 전환으로 무료 사용자 이탈 중 (대안 앱 검색 증가)
- 한국 앱스토어에서 "알약 및 약물 알림, 건강 추적기"로 표시 → 어르신 접근성 낮음

**차별 기회:**
- Medisafe의 Medfriend 콘셉트를 한국 시장에 로컬라이즈 → 검증된 수요
- Medisafe 유료 전환 시점과 맞물려 한국 무료 대안으로 포지셔닝 가능
- 한국 의약품 DB + 한국어 + 어르신 UX로 완전한 로컬라이즈

## 경쟁 환경 종합 평가

### 진입장벽 분석

| 요소 | 수준 | 근거 |
|------|------|------|
| 네트워크 효과 | **낮음** | 복약관리 앱은 개인/소규모 가족 단위 → 네트워크 효과 미미 |
| 전환 비용 | **낮음** | 복약 데이터 이전이 어렵지 않음 (약 목록 재등록 수준) |
| 데이터 해자 | **낮음** | 의약품 DB는 공공 API로 동일하게 접근 가능 |
| 브랜드 인지도 | **중간** | 파프리카케어 23만, 네이버는 브랜드 파워 있으나 복약 특화는 아님 |
| 기술 장벽 | **낮음** | CRUD + 알림 + 가족 그룹 로직, 특별한 기술 우위 불필요 |

### 대체재 위협

| 대체재 | 위협도 | 설명 |
|--------|--------|------|
| 스마트폰 기본 알람 | 중간 | 단순 시간 알림은 가능하나 복약 기록·가족 알림 불가 |
| 약 봉투/약 달력 (아날로그) | 중간 | 어르신 선호도 높으나 원격 확인 불가 |
| 스마트 약통 (IoT) | 낮음 | 한국 시장에서 보급률 극히 낮음, 가격 높음 |
| 카카오톡 메시지 | 높음 | 자녀가 직접 카톡으로 "약 드셨어요?" → 무료, 익숙, 양방향 |

### 핵심 발견

1. **"한국어 + 가족 원격 모니터링"의 조합은 국내에 부재** — Medisafe는 가족 알림이 있으나 한국어 미지원, 파프리카케어/네이버는 한국어지만 가족 원격 미지원
2. **네이버의 진입이 가장 큰 리스크** — 다만 현재는 개인 복약관리에 한정, 가족 연동 추가 여부 불확실
3. **Medisafe의 유료 전환(2026.1)은 기회** — 글로벌 시장에서 무료 대안 수요 증가, 한국 로컬 대안 포지셔닝 가능
4. **카카오톡이 가장 강력한 대체재** — "그냥 카톡하면 되잖아"를 넘어서는 자동화·기록·대시보드 가치를 제공해야 함

## 2단계 추가 조사 (2026-02-01)

### Medisafe 유료 전환 상세 (2026.1.1 시행)
- 무료 사용 시 **약 2개까지만 등록 가능**, 그 외 전면 유료 ($4.99/월, $39.99/년)
- 장기 사용자들의 강한 반발:
  - "4~5년 쓴 사용자로서 배신감" (MoneySavingExpert 포럼)
  - "만성질환자가 약 2개만 관리하라는 건 사실상 무료가 아님"
  - "앱은 '무료'라고 홍보하지만 가입 후에야 제한 확인 가능"
- 무료 대안 검색 급증: MedTimer(오픈소스), MyTherapy(완전 무료), TOM, Simpill, Apple Health Medications
- **한국 시장 시사점**: Medfriend(가족 알림) 콘셉트가 700만 사용자로 검증됨. 유료 전환에 따른 한국 로컬 무료 대안 포지셔닝 기회

### 네이버 헬스케어 복약관리 위협도 재평가: 중~낮음 (기존: 중~높음)
- 출시(2025.11.12) 후 **3개월간 가족 연동 기능 추가 없음** 확인
- 네이버의 전략 방향: B2B 생태계 집중 (임상시험 플랫폼 제이앤피메디 투자, 인바디 전략 투자, 병원·약국·보험 연계)
- "가족 연동 복약 특화"로 깊이 파고들 가능성 낮음 → 네이버앱 내 범용 건강관리 서브 서비스로 유지될 전망
- 네이버앱 알림에 묻힐 가능성, 어르신 전용 UX 부재 → 전용 앱 대비 열위

### 파프리카케어 정체 확인
- 라이프시맨틱스 인수(2024) 후 관련 뉴스 0건 (2025~2026 검색)
- 라이프레코드 플랫폼 통합 방향 → 복약관리 단독 앱 투자 불확실
- 앱스토어 평점 3.6/5 (123개), 알림 오류 리뷰 다수 → 사용자 만족도 낮음
