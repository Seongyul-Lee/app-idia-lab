## 1. 직접 경쟁자 (한국 독립 앱)

### 1-1. 유통기한 언제지
- **개발사**: (주)니즈 (Needs Co., Ltd), 서울 마포구 서울창업허브
- **플랫폼**: iOS + Android
- **주요 기능**: 바코드/영수증 스캔 자동 등록, 유통기한 임박 알림, 그룹 냉장고 공유, 다양한 관리 대상(식자재/화장품/반려동물/생활용품), SNS 로그인(카카오/네이버/구글/애플)
- **가격**: 완전 무료 (모든 기능)
- **사용자 수**: 5만 명 이상 (공식 사이트 기준)
- **최신 버전**: v1.0.0.142 (2025.10.19)
- **강점**: 바코드+영수증 OCR, 가족/그룹 공유 기능, 꾸준한 업데이트, 완전 무료
- **약점**: 화면 전환 시 로딩 느림, 레시피 추천 기능 없음, 식재료 특화보다 범용 유통기한 관리

### 1-2. 냉장고파먹기
- **플랫폼**: iOS + Android
- **주요 기능**: 재료 등록 후 만들 수 있는 레시피 추천, 카테고리별 냉장고 관리, 집밥 레시피
- **가격**: 무료
- **강점**: 레시피 추천에 특화, 사용자 리뷰 매우 긍정적 (5점 만점 다수), 자취생/맞벌이 타겟 적중, 개발팀 리뷰 소통 활발
- **약점**: 바코드 스캔 미지원 (수동 등록), 유통기한 알림 기능 약함, 소규모 개발팀으로 업데이트 지속성 불확실

### 1-3. FREEZY (프리지)
- **플랫폼**: iOS
- **주요 기능**: 바코드 스캔(사진+이름 자동), 냉장고 재고 관리, 유통기한 관리
- **가격**: 무료
- **강점**: 바코드 인식 편리
- **약점**: iOS 전용, 가족 연동 없음, 소비기한 임박 알림 미지원 (사용자 요청 다수), 위젯 없음

### 1-4. 오늘의냉장고
- **플랫폼**: iOS + Android
- **주요 기능**: 식재료 간편 추가/제거, 유효기간 확인, 보유 식재료 기반 레시피 추천(냉털 레시피), 장보기 리스트, 쿠팡 연동
- **가격**: 무료
- **강점**: 레시피 추천 + 식재료 관리 결합, 쿠팡 연동
- **약점**: 취미 개발 앱 (지속성 불확실)

### 1-5. 원더 프리지 (Wonder Fridge)
- **플랫폼**: Android
- **주요 기능**: 아이콘 기반 식품 등록, 보관 위치/수량/소비기한 관리
- **가격**: 무료
- **강점**: 직관적 아이콘 UI
- **약점**: 바코드 미지원, 소비기한 자동설정 2주 고정, 레시피 연동 없음

## 2. 간접 경쟁자 (대기업 스마트홈 플랫폼)

### 2-1. 삼성 SmartThings + 패밀리허브
- **유형**: 삼성 스마트 냉장고 전용
- **기능**: AI 비전으로 식재료 37종 자동 인식, 유통기한 설정, 레시피 추천, 원격 냉장고 확인
- **제한**: 삼성 스마트 냉장고 구매 필수 (200만 원 이상), 일반 냉장고 사용 불가

### 2-2. LG 씽큐 푸드 (ThinQ Food)
- **유형**: LG 스마트 냉장고 전용
- **기능**: AI 카메라 식품 자동 인식, 보관 목록/위치 표시, 풀무원/CJ 제휴 바코드 연동
- **제한**: LG 인스타뷰 냉장고 구매 필수, 일반 냉장고 사용 불가

## 4. 경쟁 환경 분석

### 진입장벽
- **낮음**: 기존 앱들이 대부분 소규모 독립 개발 (1-2인), 업데이트 중단/불규칙 사례 다수
- **네트워크 효과 없음**: 냉장고 관리는 개인/가정 단위 → SNS형 네트워크 효과 불필요
- **데이터 해자 미약**: 기존 앱들의 축적 데이터가 전환 장벽을 형성하지 않음

### 대체재 위협
- **스마트 냉장고**: 삼성/LG의 AI 비전 냉장고가 궁극적 대체재이나, 가격 장벽(200만 원+)으로 보급률 낮음
- **수동 관리**: 스티커, 메모, 캘린더 알림 등 아날로그 방식이 여전히 주류

### 기존 앱들의 공통 약점 (차별화 기회)
1. **등록 번거로움**: 바코드 스캔 앱도 있으나, OCR + 바코드 + 수동 입력을 모두 매끄럽게 결합한 앱은 부재
2. **레시피 + 재고 관리 통합 부족**: 둘 중 하나에만 강한 앱이 대부분
3. **가족 공유 기능 미흡**: 유통기한 언제지만 그룹 기능 제공, 나머지는 미지원
4. **푸시 알림 품질**: D-3, D-1 등 세분화된 알림 제공 앱 부족
5. **한국 식재료/레시피 DB 특화**: 해외 앱은 한국 미최적화, 국내 앱은 DB 빈약
6. **수익화 미비**: 거의 모든 앱이 무료 → 지속가능성 의문

---

## 5. 경쟁 강도 종합 평가

직접 경쟁자가 다수 존재하나, 대부분 소규모 독립 개발이며 업데이트가 불규칙하다. 네이버/카카오 등 대기업이 직접 제공하는 서비스는 없다. 삼성/LG의 스마트홈 앱은 자사 스마트 냉장고에만 한정되므로 일반 사용자 대상 직접 경쟁은 아니다. 단, "냉장고 관리 앱"이라는 카테고리 자체의 리텐션 문제(초기 등록 번거로움 → 이탈)가 근본적 도전과제다.
