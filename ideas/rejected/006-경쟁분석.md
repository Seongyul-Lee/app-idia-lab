## 1. 직접 경쟁자 분석

### 경쟁자 1: 에브리타임 (캘린더 기능)
- **플랫폼**: iOS, Android
- **핵심 기능**: 시간표 자동 연동 + 캘린더(개인 일정 관리) + 학사일정 표시 + 커뮤니티 + 강의평가
- **가격**: 무료
- **사용자 규모**: 누적 가입자 700만+, MAU 300만 (2023년 3월 기준), 잘파세대 사용자 비율 85.3%
- **수익 모델**: 대학 주변 타겟 광고, 제휴 서비스
- **강점**:
  - 대학생 10명 중 9명 이상이 사용하는 **압도적 시장 점유율** (사실상 독점)
  - 2025년 10월 캘린더 기능 정식 출시 — 시간표+개인 일정+학사일정 통합 관리 가능
  - 시간표가 캘린더에 자동 연동되는 매끄러운 UX
  - 매일 여러 번 열어보는 높은 리텐션 (커뮤니티 + 시간표 시너지)
  - 비누랩스 PM 인터뷰(2025.12)에서 "대학생 맥락에 맞는 일정 관리 기능을 계속 확장할 것"이라 명시 → 과제 추적 등 기능 확장 예고
  - 반복 일정, 컬러 구분 등 캘린더 핵심 기능 이미 구현
  - 전국 400개 캠퍼스 학교별 학사일정 제공 (일부 미지원)
- **약점**:
  - 캘린더가 아직 MVP 단계 (비누랩스 공식 인정)
  - 과제별 D-day 카운트다운, 과제 상태 추적(미시작/진행중/완료) 기능 없음
  - 과목별 과제 관리, 우선순위 분류 기능 미제공
  - 학점 시뮬레이터/이수학점 대시보드 기능 없음
  - 캘린더 기능이 범용적이며, "학사 특화" 깊이가 부족 (시험/과제/발표 등의 유형 구분 없음)
  - 커뮤니티 중심 앱이라 생산성 도구로서의 정체성은 약함

### 경쟁자 2: MyStudyLife
- **플랫폼**: iOS, Android, Web
- **핵심 기능**: 시간표 + 과제 추적 + 시험 관리 + 반복 일정 + 진행도 퍼센트 표시
- **가격**: 기본 무료, 프리미엄 구독 존재
- **사용자 규모**: 글로벌 수백만 다운로드, 한국 내 사용 미미
- **수익 모델**: 프리미엄 구독
- **강점**:
  - 학생 플래너 전문 앱 — 시간표+과제+시험 통합 관리
  - 과제 진행도 퍼센트 표시 기능
  - 웹/모바일 크로스 플랫폼 동기화
  - Google Play 리뷰: "한글화 안 되어 있지만 대학생 정도면 전혀 문제없다. 과제 항목 관리, 진행도 퍼센트 표시 좋다"
- **약점**:
  - **한국어 미지원** — 한국 대학생 대중 타겟으로는 치명적
  - 한국 학사 체계(학기제, 중간/기말, 이수학점 체계, 4.3/4.5 만점제) 미반영
  - 한국 커뮤니티·입소문 기반 부재
  - 한국 대학 학사일정 데이터 연동 불가
  - 에브리타임 시간표와 연동 불가

### 경쟁자 3: 학교 플래너(School Planner / daldev)
- **플랫폼**: Android (iOS 없음)
- **핵심 기능**: 시간표 + 과제 관리 + 시험 관리 + 성적 계산
- **가격**: 무료 (광고 기반)
- **사용자 규모**: Google Play 1,000만+ 다운로드 (글로벌), 한국 사용자 일부
- **강점**:
  - 시간표, 과제, 시험, 성적을 한 앱에서 관리
  - 과제 마감일, 우선순위 정렬 가능
  - 다국어 지원 (한국어 포함)
- **약점**:
  - **광고가 매우 많음** ("시간표 한번 보려면 광고 하나 봐야 됨" — Google Play 리뷰)
  - iOS 미지원 → 한국 대학생 iPhone 비율(약 60%+) 고려 시 타겟의 절반 이상 커버 불가
  - 한국 학사 체계 특화 아님
  - UX/디자인이 오래됨
  - 에브리타임 연동 불가

## 2. 간접 경쟁자 / 대체재

| 대체재 | 사용 방식 | 한계 |
|--------|----------|------|
| 노션(Notion) | 직접 템플릿을 만들어 과제/시험 관리 | 학사 특화 아님, 세팅에 시간 소요, 알림 약함 |
| 구글 캘린더 | 일정 수동 입력 + 알림 | 과목별 과제 추적 불가, D-day 없음, 학점 관리 불가 |
| 네이버 캘린더 | 일정 관리 + 한국 공휴일 | 학사 특화 기능 전무 |
| 잡코리아/사람인 학점 계산기 | 웹에서 학점 환산 | 일정 관리와 분리, 앱이 아님, 일회성 사용 |
| LMS (e-class 등) | 과제 확인 + 제출 | 학교별 분산, 알림 불안정, 통합 뷰 없음 |

## 3. 진입장벽 및 위협 평가

### 에브리타임의 네트워크 효과 / 전환 비용
- **매우 높음**. 에브리타임은 대학생 생태계를 사실상 독점하고 있으며, MAU 300만은 한국 전체 대학 재적학생 300만 명과 거의 동일
- 대학생이 "또 다른 앱"을 설치해야 하는 전환 비용이 높음
- 에브리타임에서 커뮤니티+시간표+캘린더를 이미 사용 중이므로, 일정 관리만을 위해 별도 앱을 쓸 동기가 약함
- **핵심 위협**: 에브리타임이 캘린더 MVP를 이미 출시했고, 과제 추적·학사 특화 기능 확장을 공식 예고함. 차별화 포인트였던 기능들이 에브리타임에 흡수될 가능성 높음

### 데이터 해자
- 에브리타임: 전국 400개 캠퍼스 학사일정 데이터, 강의 데이터베이스, 강의평가 데이터 보유
- 신규 앱이 이 수준의 대학별 데이터를 구축하는 것은 현실적으로 매우 어려움

### 대기업 위협
- 에브리타임(비누랩스)은 직접적 대기업은 아니나, 대학생 시장에서 대기업급 지배력 보유
- 네이버/카카오가 이 니치에 직접 진출할 가능성은 낮으나, 에브리타임의 기능 확장이 사실상 대기업 수준의 위협

## 4. 차별화 가능성 평가

### 학사플래너가 제안하는 차별점 vs 현실
| 차별점 | 실현 가능성 | 지속 가능성 |
|--------|------------|------------|
| 과제별 D-day + 맞춤 알림 | 기술적으로 가능 | 에브리타임이 동일 기능 추가 시 소멸 (예고됨) |
| 과제 상태 추적(미시작/진행중/완료) | 기술적으로 가능 | 에브리타임 확장 시 소멸 가능 |
| 학점 시뮬레이터 | 기술적으로 가능 | 니치 기능, 사용 빈도 낮음 (학기당 1-2회) |
| 공강 시간 과제 자동 제안 | 기술적으로 가능 | 독자적 차별점이 될 수 있으나, 단독으로 앱 설치를 유도하기 어려움 |

### 결론
- 에브리타임이 캘린더를 MVP로 출시하고 적극 확장을 예고한 상황에서, **학사플래너가 제시하는 거의 모든 핵심 기능이 에브리타임에 흡수될 위험이 매우 높음**
- 학점 시뮬레이터는 차별점이나, 이것만으로는 별도 앱 설치의 동기가 되기 어려움 (웹 학점 계산기가 이미 다수 존재)
- "에브리타임 없는 대학생활"이 사실상 불가능한 현실에서, 일정 관리를 위해 별도 앱을 추가 설치할 사용자 비율은 극히 제한적
