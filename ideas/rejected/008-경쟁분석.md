# 008-구독누수방지매니저 경쟁 분석

## 분석일: 2026-02-01

---

## 1. 직접 경쟁자

### 1-1. 왓섭 (Whatssub) — 가장 직접적인 경쟁자
- **유형**: 한국 구독 관리 전용 앱 (스타트업)
- **설립**: 2020년, 대표 김준태
- **투자**: 끌림벤처스, 소프트뱅크벤처스, KB인베스트먼트에서 프리시리즈A 유치
- **마이데이터 본허가**: 금융위원회로부터 본인신용정보관리업(마이데이터) 본허가 획득
- **주요 기능**:
  - 카드·계좌·네이버페이 연동으로 구독 자동 탐지
  - 공동인증서 기반 모든 금융사 연결
  - 앱 내에서 구독 서비스 직접 해지 가능
  - 정기결제 + 생활 지출 통합 관리
  - AI 에이전트 기반 자산관리로 진화 중 (2025.12 금융사 밋업 발표)
- **비즈니스 모델**: 사용자 무료, B2B 데이터 처리 및 분석 대행 수수료
- **강점**: 마이데이터 인허가 기반 자동 구독 탐지, 해지 대행, 금융사 임베드 전략
- **약점**: 인지도 제한적 (메이저 앱 대비), 사용자 규모 미공개

### 1-2. 뱅크샐러드
- **유형**: 종합 자산관리 앱 (마이데이터 기반)
- **사용자**: 수백만 MAU 추정
- **구독 관련 기능**:
  - 마이데이터 기반 카드 내역에서 정기결제 자동 추출
  - 가계부 기능 내 구독 지출 확인 가능
- **강점**: 대규모 사용자 기반, 종합 자산관리에 구독 관리가 부가 기능으로 포함
- **약점**: 구독 전용 기능이 아닌 부가 기능 수준. 카테고리 분류, 미사용 추적, 절약 리포트, 해지 가이드 미제공
- **위협**: 언제든 구독 관리 전용 탭을 강화할 수 있는 기술력과 사용자 기반 보유

### 1-3. 토스
- **유형**: 종합 금융 슈퍼앱
- **사용자**: 2,000만+ MAU
- **구독 관련 기능**:
  - '정기결제 조회' 기능으로 카드/계좌 기반 구독 자동 감지
  - 토스뱅크 블로그에서 "구독 서비스 이렇게 사용하면 돈 낭비" 콘텐츠 발행
- **강점**: 압도적 사용자 기반, 마이데이터 인허가 보유, 금융 서비스와 자연스러운 통합
- **약점**: 구독 관리 전문 기능(카테고리 분류, 미사용 추적, 절약 리포트) 미제공
- **위협**: 핵심 기능으로 구독 관리를 강화할 경우 독립 앱의 존재 이유가 사라짐

## 2. 간접 경쟁자 (글로벌)

### 2-1. Rocket Money (구 Truebill) — 미국
- 500만+ 회원, 은행 연동 기반 구독 자동 감지 + 대행 해지
- "첫 주에 $200 이상 절약" 마케팅
- 한국 미진출, 한국 결제 시스템 비호환

### 2-2. Bobby / Subby — 글로벌
- 수동 등록 방식 구독 트래커
- 한국 로컬라이징 없음 (원화, 한국 서비스 DB 미지원)
- 소규모 유틸리티 앱 수준

## 3. 경쟁 구조 분석

### 진입장벽
| 장벽 유형 | 수준 | 설명 |
|-----------|------|------|
| 마이데이터 인허가 | **높음** | 왓섭은 마이데이터 본허가 보유. 1인 개발자가 취득 불가능 |
| 네트워크 효과 | **높음** | 토스(2,000만+), 뱅크샐러드(수백만) 사용자 기반 |
| 데이터 해자 | **높음** | 기존 앱들이 축적한 결제 내역 데이터 |
| 전환 비용 | **낮음** | 수동 등록 앱으로의 전환 비용은 낮지만, 자동→수동 전환은 UX 하락 |

### 대체재 위협
- **앱스토어/구글플레이 자체 구독 관리**: iOS 설정 > 구독, Google Play > 정기결제에서 기본 관리 가능
- **카드사 앱**: 카드 앱에서 정기결제 내역 확인 가능
- **엑셀/노션 수동 관리**: 일부 사용자는 이미 노션 등으로 구독 관리 중 (네이버 블로그 사례)

## 4. 핵심 결론

**구독 관리 전용 앱 시장에서 왓섭이 이미 마이데이터 인허가를 확보하고 자동 감지 + 해지 대행까지 제공 중**이며, 토스/뱅크샐러드가 부가 기능으로 정기결제 조회를 무료 제공하고 있다. 수동 등록 방식의 독립 앱이 이들과 경쟁하여 유의미한 사용자를 확보하기는 매우 어려운 구조다.
