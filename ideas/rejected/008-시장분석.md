## 1. 한국 구독 경제 시장 규모

### 전체 구독 경제
- **2025년 국내 구독 시장**: 약 100조원 규모 (KT경제경영연구소, SK텔레콤 전망)
- **2020년**: 49조원 → **2025년**: 100조원 (연평균 15% 성장)
- 디지털 콘텐츠 + 식품 + 패션 + 가전 등 전 산업으로 확산

### 디지털 구독 서비스 이용 현황
- 한국인 1인당 평균 **7.3개** 구독 서비스 이용 (한국 소비자원, 2025)
- 월평균 구독 지출: **12만원 이상** (한국 소비자원)
- 서울시 조사: 2024년 구독경제 이용률 전년 대비 3.8배 증가, 월평균 지출 40,530원
- OTT 1인당 평균 2.1개 구독, 월 이용료 12,005원
- 월 구독료 분포: 3만원 미만 30.5%, 3~5만원 22.9%, 5~10만원 22.3%, 10~15만원 9.4%, 15만원 이상 15.0%

### 한국 OTT 시장
- OTT 가입자 약 1,700만명 (중복 포함)
- 쿠팡 유료회원 1,400만명, 넷플릭스 이용자 1,125만명

## 2. 구독 관리 소프트웨어 글로벌 시장

- 글로벌 구독 빌링 관리 시장: **2025년 USD 8.47~10.86B** → 2032~2035년 USD 17~37B (CAGR 13~16%)
- 개인용 구독 트래커 시장은 별도 통계 부재 (B2B SaaS 빌링 관리가 주류)

### Rocket Money (미국 레퍼런스)
- 500만+ 회원
- 은행 연동 기반 자동 구독 감지 + 대행 해지 + 요금 협상 서비스
- 프리미엄 구독 기반 수익 모델
- "첫 주에 $200 절약" 홍보

## 3. TAM → SAM → SOM 추정

### TAM (총 시장 규모)
- 한국 디지털 구독 서비스 이용자: 약 **2,000만명+**
- 연간 구독 지출 총액: 2,000만명 × 월 5만원(중간값) × 12개월 = **약 12조원/년**

### SAM (유효 시장 규모)
- 다수(5개+) 구독을 이용하며 관리 니즈가 있는 20~40대: 약 **500~700만명**
- 이 중 별도 구독 관리 앱을 설치할 의향이 있는 비율: 추정 10~20%
- SAM: **50~140만명**

### SOM (확보 가능 시장)
- 토스/뱅크샐러드가 무료로 기본 관리 제공하는 상황
- 왓섭이 마이데이터 기반 전문 서비스 제공 중
- 수동 등록 방식 신규 앱이 확보 가능한 사용자: **극히 제한적 (1~3만명 수준)**
- 이유: 자동 감지 대비 수동 입력의 UX 열위, 기존 무료 대안의 존재

## 4. 고객 직무(JTBD) 분석

### 핵심 직무
> "내가 매달 어디에 얼마를 쓰고 있는지 한눈에 파악하고, 안 쓰는 구독을 놓치지 않고 해지하고 싶다"

### 기존 대안의 충족 수준
| 대안 | 직무 충족도 | 과잉/부족 |
|------|------------|-----------|
| 토스 정기결제 조회 | 중 | 파악은 자동이나 해지·분석은 부족 |
| 뱅크샐러드 | 중 | 지출 분석에 구독 포함되나 전용 관리 미흡 |
| 왓섭 | 고 | 자동 탐지 + 해지 대행 + 관리 전용 |
| iOS/Android 설정 | 하 | 앱스토어 결제만 확인, 카드 직접 결제 불포함 |
| 수동 관리 (노션/엑셀) | 하 | 입력 부담 크고 알림 없음 |

### 네이버 커뮤니티 불만·수요 근거
1. **무료 체험 후 자동 결제 함정**: "밀리의 서재 1달 무료체험 후 자동 결제가 되어서" (네이버 지식iN), "무료 체험 후 유료로 전환되는 앱은 미리 관리하지 않으면 낭비" (네이버 블로그)
2. **해지 방법 복잡**: "정기결제 취소가 안 돼요" (네이버 지식iN — 9,772건 관련 질문), 앱마다 해지 경로가 다름
3. **지출 파악 어려움**: "이게 왜 결제됐지?" (네이버 블로그), 여러 카드에 분산된 정기결제
4. **구독 피로감**: "부담스러운 구독시대" (네이버 카페), "안쓰는 구독 정리" 관련 블로그 게시글 106,000+건

### Pain Point 존재 여부
**Pain Point 자체는 매우 명확하고 실재한다.** 그러나 이 문제를 해결하는 기존 대안(특히 왓섭, 토스, 뱅크샐러드)이 이미 시장에 존재하며, 수동 등록 방식으로는 기존 대안 대비 UX 열위.
