# 살았니 (SafePin)

## 한줄 요약
> 1인가구를 위한 일일 안부 체크인 앱 — 매일 버튼 하나로 "나 살아있어"를 확인하고, 미응답 시 긴급연락처에 자동 알림을 보낸다.

## 해결하려는 문제 (Pain Point)
- **혼자 쓰러지면 아무도 모른다는 불안**: 한국 1인가구 비율이 36%를 돌파했고, 서울은 거의 50%에 육박한다. 1인가구 62%가 외로움을 느끼며(Korea Herald 2025), "혼자 살다 쓰러지면 아무도 모를까 봐 불안하다"는 게시글이 네이버 블로그/카페에서 지속적으로 발견된다.
  - 블로그 근거: "쓰러지면 끝? 1인 가구가... 혼자 사는 집 필수 안전..." (blog.naver.com/khun0828, 2026.01.18)
  - 카페 근거: "혼자 사는 삶에는 외로움, 불안, 응급상황의 어려움 등 여러 문제가 뒤따릅니다" (은퇴 후 50년 카페 설문조사)
  - 실제 사례: "동생은 인천에 혼자 살다보니 경찰에 신고했고... 들어가니 동생은 침대에 엎드려 의식이 없었고" (레몬테라스 카페)
- **고독사 75%가 40~60대**: 고독사는 노인만의 문제가 아니라 중장년 1인가구의 핵심 리스크다 (ZDNet 2026.01.05).
- **기존 정부 서비스의 한계**: 한국 지자체별로 AI 안부확인 서비스(한전 전력데이터 기반, SKT/KT AI 전화 등)가 운영되지만, 대상이 고령층/취약계층에 한정되고, 신청 절차가 복잡하며, 지자체별로 파편화되어 일반 청장년 1인가구는 사각지대에 놓여 있다.
- **글로벌 수요 검증**: 중국 "死了麼(Are You Dead?)" 앱이 2026년 1월 애플 앱스토어 유료 1위를 기록하며 바이럴됨. CNN, BBC, Reuters, The Guardian 등 주요 외신이 일제히 보도. 사용자가 48시간 이내 체크인하지 않으면 긴급연락처에 자동 알림을 보내는 단순한 구조로, "1인가구 안전 체크인"이라는 니즈가 전 세계적으로 검증되었다.

## 타겟 사용자
- **주요 타겟**: 20~50대 1인가구 (직장인, 자취생, 프리랜서, 이혼/별거 중장년)
- **연령대**: 20~59세 (기존 정부 서비스가 커버하지 못하는 연령층)
- **규모 추정**: 한국 1인가구 약 1,000만 가구 중 20~59세 약 600만+. 타겟 전환율 1% 가정 시 6만 사용자 확보 가능.

## 핵심 기능 (MVP 범위, 최대 3개)
1. **일일 체크인**: 설정한 시간에 푸시 알림 → 버튼 하나로 "살았어" 체크인. 일정 시간(예: 12시간/24시간/48시간 선택 가능) 미응답 시 긴급연락처에 자동 SMS/카카오 알림 발송.
2. **긴급연락처 관리**: 가족, 친구, 이웃 등 최대 3명의 긴급연락처 등록. 미응답 알림 수신 및 사용자 위치(마지막 체크인 위치) 공유.
3. **안부 타임라인**: 체크인 기록 달력 뷰 — 연속 체크인 스트릭, 한줄 상태 메모(선택). 긴급연락처가 실시간으로 안부 상태를 확인할 수 있는 대시보드.

## 수익 모델
- **프리미엄 구독 (월 1,900원~3,900원)**: 체크인 횟수 확대(1일 3회), 위치 공유, 카카오톡 알림 연동, 체크인 리포트
- **1회성 유료 활성화 (1,200원)**: 중국 死了麼 모델 참고 — 앱 설치 후 긴급알림 기능 활성화에 소액 결제
- **B2G(정부 연계)**: 지자체 고독사 예방 사업 파트너십 (장기)

## 기술 구현 검토
- **React Native + TypeScript**: 완전 구현 가능. 푸시 알림(expo-notifications 또는 react-native-push-notification), 백그라운드 스케줄링, 위치 서비스 등 필요 라이브러리 모두 RN 생태계에서 지원됨.
- **Supabase**: 사용자 인증, 체크인 기록 저장, 긴급연락처 관리, 실시간 상태 구독(Realtime) 모두 커버 가능. Edge Functions로 미응답 감지 및 알림 트리거 로직 구현 가능. Cron Job(pg_cron)으로 체크인 미응답 감지 스케줄링.
- **외부 서비스 의존도**:
  - SMS 발송: Supabase Edge Functions + 알리고/NHN Cloud 문자 API (건당 약 20원)
  - 카카오 알림톡: 카카오 비즈메시지 API (건당 약 8원) — 프리미엄 기능
  - 푸시 알림: FCM (무료)
  - 결제: 인앱결제 (App Store / Google Play 기본 제공)
  - 본인인증: 불필요 (전화번호 기반 OTP로 충분)
