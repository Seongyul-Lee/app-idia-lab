# 005-절약챌린지 경쟁분석

## 조사일: 2026-01-31

---

## 1. 직접 경쟁자 분석

### 경쟁자 1: 뱅크샐러드 '샐러드게임'
- **플랫폼**: iOS, Android (뱅크샐러드 앱 내 기능)
- **핵심 기능**: 5일간 5명 팀 구성 → 팀 예산(25만 원) 지켜쓰기 → 성공 시 지출액 100% 상금 환급. 협동 미션, 팀 채팅, 뿅망치(지출 통제 유도) 기능
- **가격**: 무료 참가 (참가권 필요, 뽑기로 획득)
- **사용자 규모**: 1회차 5,000명 참여, 정규 편성 후 지속 운영
- **수익 모델**: 뱅크샐러드 본체의 금융상품 추천 수수료. 샐러드게임 자체는 앱 유입/리텐션 수단
- **강점**:
  - 마이데이터 기반 자동 지출 연동 (수동 입력 불필요)
  - 실질 상금 제공 (최대 54만 원/팀)
  - 뱅크샐러드 기존 사용자 풀 활용
  - 팀 협동 게이미피케이션으로 높은 참여율 (앱 유입률 108% 증가)
  - 참여자 평균 14만 원 절약 효과 검증
- **약점**:
  - 뱅크샐러드 앱 내 부가 기능으로, **독립 앱이 아님** → 절약 챌린지만을 위한 진입 장벽 높음
  - 참가권 뽑기 방식 → 원하는 시점에 참여 불가
  - 5일 단위 단기 게임만 가능 → 장기 습관 형성에 한계
  - 무지출 스트릭, 캘린더 등 **개인 기록 관리 기능 부재**
  - 오픈 커뮤니티(절약방) 기능 없음 → 게임 외 소셜 교류 제한
  - KCB 데이터 연동 필수 → 진입 허들

### 경쟁자 2: 카카오페이 '카드 절약 챌린지'
- **플랫폼**: 카카오톡/카카오페이 앱 내 기능
- **핵심 기능**: 마이데이터 카드 연동 → 전월 동일 일자 대비 절약 시 '승리 배지' 획득 → 배지 4개 모으면 리워드 포인트 분배
- **가격**: 무료
- **운영 방식**: 매월 1일 시작, 주 단위 배지 획득, 다음 달 12일 리워드 지급
- **수익 모델**: 카카오페이 결제/금융 생태계 활성화 마케팅 도구
- **강점**:
  - 카카오톡 4,700만+ 사용자 기반으로 접근성 극대화
  - 마이데이터 자동 연동으로 수동 입력 불필요
  - 또래 비교 기능 (연령대별 소비 비교)
  - 지인 공유 시 보너스 배지 제공 → 바이럴 장치
- **약점**:
  - **독립 서비스가 아닌 프로모션 수준** → 기능 깊이 제한
  - 커스텀 챌린지 생성 불가 (전월 대비 절약만 가능)
  - 커뮤니티/소셜 기능 없음
  - 무지출 캘린더, 스트릭, 배지 시스템 등 게이미피케이션 깊이 부족
  - 리워드가 전체 참여자 N분의 1 → 개인당 실질 보상 미미
  - 카카오페이 카드 사용자만 참여 가능

### 경쟁자 3: 챌린저스 (화이트큐브)
- **플랫폼**: iOS, Android (독립 앱)
- **핵심 기능**: 다양한 챌린지 참여 (돈 걸고 미션 수행, 성공 시 환급 + 추가 상금). 2024년부터 뷰티 커머스로 피봇
- **사용자 규모**: 누적 171만 명 (2024년 3월 기준)
- **매출**: 2024년 연매출 150억 원 (전년 대비 2.5배 성장)
- **수익 모델**: CPA/CPS 광고 (브랜드 제휴 챌린지) + 뷰티 커머스 수수료
- **강점**:
  - 챌린지 앱 1위 (건강/습관 카테고리)
  - 검증된 '돈 걸기' 게이미피케이션 모델
  - 광고주 재계약률 62%로 수익 모델 안정화
  - 171만 사용자 기반
- **약점**:
  - **뷰티 커머스로 피봇** → 순수 절약/무지출 챌린지 비중 대폭 감소
  - 앱 정체성이 '뷰티 득템 앱'으로 전환 → 절약 목적 사용자 이탈 가능
  - 무지출 스트릭/캘린더 등 일상 절약 기록 기능 없음
  - 커뮤니티(절약방) 기능 미흡
  - 매일의 절약 습관 추적보다 단기 챌린지 이벤트 중심

## 3. 진입장벽 분석

### 기존 앱의 네트워크 효과
- **뱅크샐러드**: 마이데이터 기반 자산 연동이 핵심 해자. 다만 샐러드게임은 부가 기능 수준
- **카카오페이**: 카카오톡 생태계의 압도적 접근성. 그러나 절약 챌린지는 프로모션 수준
- **챌린저스**: 171만 사용자, 그러나 뷰티 커머스로 피봇하여 절약 전문성 약화

### 전환 비용
- **낮음**: 기존 서비스들이 절약 챌린지를 핵심 가치로 제공하지 않으므로, 사용자가 별도 전용 앱을 설치하는 데 장벽이 낮음
- 거지방/인스타 사용자는 특별한 전환 비용 없이 병행 사용 가능

### 데이터 해자
- **뱅크샐러드/토스**: 마이데이터 연동으로 강력한 데이터 해자 보유
- **절약챌린지(우리 앱)**: 마이데이터 없이 수동 입력 기반이므로 데이터 해자 구축 어려움. 대신 **커뮤니티와 게이미피케이션 경험**이 차별화 해자가 되어야 함

### 대체재 위협
- **중간**: 카카오톡 거지방이 무료 대안으로 존재하나, 체계적 관리 불가라는 명확한 한계 보유
- 뱅크샐러드/카카오페이가 챌린지 기능을 독립 앱 수준으로 강화할 경우 위협적

## 4. 차별화 전략

### 005-절약챌린지의 포지셔닝
> **"무지출 챌린지 전용 앱"** — 가계부도, 금융 슈퍼앱도, 뷰티 커머스도 아닌, 오직 절약 습관 형성과 커뮤니티에 집중하는 앱

### 핵심 차별점
| 차별점 | 경쟁사 대비 |
|--------|------------|
| **극단적 단순화** (하루 한 번 탭 인증) | 뱅크샐러드/토스의 복잡한 UI 대비 |
| **스트릭 + 캘린더** 기반 시각적 동기부여 | 챌린저스의 단기 이벤트 대비 장기 습관 형성 |
| **앱 내 절약 커뮤니티(절약방)** | 카카오톡 거지방의 비체계적 소통 대비 |
| **커스텀 챌린지 생성** | 카카오페이의 단일 챌린지 유형 대비 |
| **독립 전용 앱** | 뱅크샐러드/카카오페이의 부가 기능 대비 |

### 리스크
- 마이데이터 미연동으로 자동 지출 추적이 불가 → 수동 입력의 번거로움 (단, "하루 한 번 탭"으로 극단적 단순화로 대응)
- 커뮤니티 콜드 스타트 문제 → 카카오톡 거지방 사용자를 타겟 마케팅으로 유입 필요
